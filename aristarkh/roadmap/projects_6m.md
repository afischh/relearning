# Проекты на ближайшие 6 месяцев (расширенная версия)

Этот план не является жёстким расписанием.
Это карта возможных работ, которые:
- обучают,
- дают реальные навыки,
- могут перерасти в практическую пользу и доход.

Темп: медленный, устойчивый, без перегруза.

---

## Исходная оценка уровня

На текущий момент пользователь:

- программирует около нескольких недель;
- уверенно работает с git, VS Code, Python-скриптами;
- понимает архитектурные различия (build-time / run-time);
- уже интегрировал ИИ в реальный проект;
- мыслит архитектурно, несмотря на слабую математическую базу.

Это уровень:
**начинающий разработчик + сильный архитектор намерения**.

## Проект A. quiet_logos v1.2 — агент как исследователь

**Язык:** Python  
**Фокус:** ИИ-оркестрация, экономия ресурсов, ответственность

### Цели
- Сделать комментарии Аристарха более аналитическими:
  - темы,
  - повторяющиеся понятия,
  - изменения тона дневника.
- Добавить режимы:
  - `analysis` (короткий),
  - `reflection` (философский),
  - `indexing` (ключевые темы).

### Практика
- чтение существующего кода,
- минимальные правки,
- логирование стоимости и времени генерации.

## Проект B. Flutter Writer v0.3 — GUI для дневника

**Язык:** Dart / Flutter  
**Фокус:** пользовательский интерфейс, форма, управление процессом

### Цели
- GUI-программа (desktop):
  - ввод записи (date / title / quiet / tech),
  - кнопка «Сохранить»,
  - кнопка «Собрать сайт»,
  - кнопка «Открыть сайт».
- Минимальный дизайн:
  - один экран,
  - никакой сложности,
  - стабильность важнее красоты.

### Польза
- уменьшение трения,
- ощущение “прибора”, а не кода,
- шаг к Pocket AI Lab.

## Проект C. GUI-парсер онлайн-библиотек

**Язык:** Python + GUI (Flutter или Tkinter)  
**Фокус:** исследование текстов, данных, культуры

### Цели
- Программа для:
  - поиска книг и статей в онлайн-библиотеках,
  - загрузки метаданных,
  - извлечения аннотаций и оглавлений.
- Возможность:
  - фильтрации по темам,
  - экспорта в JSON/CSV.

### Навыки
- HTTP-запросы,
- парсинг HTML,
- работа с данными,
- простая визуализация.

**Фокус:** исследование дискурсов и нарративов
