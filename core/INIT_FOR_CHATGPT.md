# INIT_FOR_CHATGPT.md

## Назначение

Этот файл — точка инициализации контекста для нового чата ChatGPT,
работающего с проектом **relearning**.

ChatGPT **не имеет прямого доступа** к репозиторию.
Весь контекст передаётся через этот файл и дополнительные фрагменты,
которые я (автор проекта) предоставляю вручную.

Если ты читаешь этот файл — считай, что тебе передан базовый доступ
к архитектуре, философии и текущему состоянию проекта.
